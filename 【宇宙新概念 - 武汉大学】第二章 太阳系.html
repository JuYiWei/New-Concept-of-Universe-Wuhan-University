一、太阳

1.1 天文单位

1天文单位 = 1 光年 = 1.495979 x 10^8 千米
注：光 1 年传播的距离

1.2 太阳的基本参数

半径：6.9599 x 10^5 千米
质量：1.99 x 10^30 千克
位置：银心 3.26 万光年，银面以北 26 光年
运动：绕银心转动，同时相对其他恒星平动
太阳常数：能量 0.136 焦/(s*cm^2)

1.3 太阳的大气

光球层：5770K，存在太阳黑子
色球：可见光少，不容易观测
日冕：最外层，直径可达太阳的10-20倍，200万K，物质呈电离状态

1.4 日球

太阳为中心，太阳风及磁场延伸的空间范围

1.5 太阳能量

氢核聚变

更热的恒星：碳氮循环

1.6 中微子

不带电荷，没有静止质量，真空光速运动，微弱的相互作用
实际测得的数量不足理论模型的 1/3

二、地球和月球

2.1 地球

  • 地球基本参数
椭球体：长轴 d = 6378km、短轴 c = 6357km
平均半径：6371km
质量：5.976 x 10^24kg
平均密度：5.517 g/cm3
年龄：46亿年
  • 地球主要分层
地核：内核 + 外核
地幔：约 2900km
地壳（地表）：平均约 17km
  • 大气
质量：5.13 x 10^18kg
质量分布：氮气：75% 氧气：23%
容积分布：氮气：78% 氧气：21%
  • 大气分层
对流层：生活区，有天气变化
平流层：飞机飞行
电离层：
  • 地球自转、公转和地轴进动
自转：23 时 56 分 4 秒
公转：365.24219 天
地轴进动：地轴绕黄轴的圆锥形运动，25800 年

2.2 月球

  • 月球的基本参数
月底距离：max = 406700km、min = 356400km、avg = 38400km
半径：1738.2km
质量：7.35 x 10^22kg
平均密度：3.34g/cm3
引力：地球的 六分之一
  • 盈亏现象
原因：1）本身不发光，反射太阳光；2）日月地地三者位置不断变化；
  • 月球表面
暗的：平坦
亮的：高低不平
没有大气，昼夜温差大

2.3 日月地地天文现象

潮汐：月球引力的影响
日食：日月地
月食：日地月

2.4 日地空间

太阳和地球之间的空间，太阳主导，影响地球

三、太阳系的其他天体

3.1 八大行星

水金地火木土天海
根据质量、大小和化学组成分类：
类地行星：水金地火（密度大，铁镍核）
巨行星：木土（氢氦）
远日行星：天海

除最近的水星和最远（被移除的冥王星）行星公转轨道偏心率都很小
还得靠相对论才能解释

3.2 小天体

小行星、彗星、流星
  • 提丢斯-波得定则
设：土星到太阳的距离为 100 个单位
行星距离：R = 0.4 + 0.3 x 2^(n-2) 个单位
  • 小行星带
也符合提丢斯定则 n = 5

来源：
1）存在大行星，后来爆炸了
2）原始星云演化不健全，没有形成大行星
  • 彗星
靠近太阳有尾巴
哈雷彗星：被预言的周期性彗星，76-79年？
  • 流星和陨星
流星：质量小，划过天空就没了
陨星：质量大的，落到地面
流星雨：周期性的较多的
  • 行星新定义
1）位于围绕太阳的轨道上
2）有足够大的质量来克服固体应力，达到流体静力平衡
3）并可以清空自身轨道附近区域的天体

不满足条件 3 的称为矮行星
